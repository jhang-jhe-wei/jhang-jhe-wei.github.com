LINE 聊天機器人—台科不分系學姊

「台科不分系學姊」是一款專為台科全校不分系新生製作的 LINE 聊天機器人，為了解決每年全校不分系新生入學時的困惑，因此由本人主動發想、設計並完成此專案，目前提供選課解答、畢業條件和校內大小事通知等功能。

Sketchup Rebar Helper Plugin

Sketchup 是建築界常用的 3D 建模軟體，此案的需求為結構體內自動部屬鋼筋，傳統情況下，建模者須依照結構體尺寸、包層和鋼筋種類建計算出並手動建置樣本鋼筋，之後不斷複製樣本鋼筋到特地位置，工作內容十分重複且費時，使用此插件後，藉由輸入參數，便可直接產生出相符於結構體的鋼筋。

廣吉系統

此專案是廣吉食品的 C2M 計畫，整合 ERB 系統、配方分析、物流分析、商品與論爬蟲和精準導購。在爬蟲部分可以搜集電商（如蝦皮、樂天）和論壇（PTT、Dcard）有關於麥片商品的所有資訊。

聊天機器人群組管理系統

此案的需求為建立一個 LINE Chatbot 後台管理系統，此後台會記錄含有此機器人的群組，可為每個群組加上標籤，核心功能為群發，可指定標籤及群組，傳送內容包含文字和檔案，若傳送檔案為pdf，會轉換成圖片後加上浮水印再傳至群組聊天室。

留言板系統

此作品是網頁安全的期中作業，在不使用任何框架和第三方套件的前提下製作一個留言板系統，並需要防禦其他人的攻擊，在其中本人製作了一個簡易的 MVC 架構，並且符合 Restful API 的設計原則。
